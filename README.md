Boston Crime Report
===================

### Problem
Given latitude, longitude, and radius, find all the crimes in that area.

### Input
`./crimefinder <PATH_TO_JSON_DATABASE> <LATITUDE> <LONGITUDE> <RADIUS>`

### Output
```
<COMPNOS1>: <DISTANCE>
<COMPNOS2>: <DISTANCE>
...
```
